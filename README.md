# STUDENTS MANAGEMENT SYSTEM

## Overview

On this occasion, the task was to develop a Python-based program that would allow school administrative assistants to register students in their systems.

## Requirements

The user requested the following for the registration and management system:

1. That the system would register new students consecutively as required.
2. Display the list of students stored in the main data structure (list).
3. The system should be able to search for and find a student when their name or ID is entered.
4. That the system can delete and update the data of a student according to their name.
5. The system should be able to locate the student's name and subsequently delete all their information from the main list.
